# PapaHut

A full stack, fully responsive, React.js pizza customization & ordering web application for pizza lovers! Inspired by a combination of Pizza Hut & Papa John's!

## Technologies Used

- React.js
- Node
- Express
- PostgreSQL
- HTML5
- CSS3
- PM2
- AWS S3
